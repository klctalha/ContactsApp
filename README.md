# Contacts Application
Bu proje kullanıcıların kişilerini yönetebileceği, Jetpack Compose ile geliştirilmiş modern bir Android rehber uygulamasıdır. 
Proje MVVM mimarisi, Room veritabanı ve Kotlin'in en güncel kütüphaneleri kullanılarak geliştirilmiştir.

## Kullanılan Teknolojiler & Kütüphaneler
* Kotlin
* Jepack Compose
* MVVM
* Room Database
* Coroutines
* Coil
* Navigation
* Gson
* ViewModelFactory
* Material3 Design Components

## Özellikler
Kişi Yönetimi: Yeni kişi ekleme, mevcut kişileri düzenleme, görüntüleme ve silme
Arama Fonksiyonu: İsim üzerinden hızlı ve canlı arama
Modern UI: Jetpack Compose ile responsive ve kullanıcı dostu arayüz tasarımı
Veri Doğrulama: Hatalı veri girişlerini engelleyen (telefon hane sayısı, e-posta formatı vb.) merkezi validasyon sistemi
Kullanıcı Deneyimi: Bağlamsal menü ile hızlı işlemler

## Mimari Yapı (MVVM)
View: UI bileşenleri ve Compose ekranları
ViewModel: İş mantığı ve UI durum yönetimi
Repository: Veri kaynakları (Room) ve UI arasındaki köprü
Entity: Veritabanı modeli
